# Lector QR: lector_qr
Detecta códigos QR y muestra su validez mediante LEDs de colores

## Dependencias
Zbarcam
Use apt-get install zbarcam para instalar

## Uso
Ejecute el archivo servicio1.py utilizando
python servicio1.py
